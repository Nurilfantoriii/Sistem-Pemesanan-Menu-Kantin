# Sistem-Pemesanan-Menu-Kantin
Tugas Besar Pemrograman Web (PWEB) - Aplikasi Sistem Pemesanan E-Kantin Mahasiswa berbasis PHP Native (PDO) dan Bootstrap 5. Dikembangkan oleh Kelompok 11 Informatika UPN "Veteran" Jawa Timur. Fitur utama mencakup manajemen CRUD menu oleh admin, pelacakan status pesanan secara real-time, sistem arsip antrean, dan berbagai opsi pembayaran cashless
